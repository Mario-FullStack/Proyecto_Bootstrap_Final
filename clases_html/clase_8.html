Helpers

Estas clases son usadas para ocultar o mostrar contenido via media queries en dispositvos. Usa una o varias combinaciones de las clases disponibles para alternar contenido por los diferentes viewport breakpoints.

Clearfix

Borra rápida y fácilmente el contenido flotante dentro de un contenedor.

              
 <div class="bg-info clearfix">
   <button type="button" class="btn btn-secondary float-start">Example Button floated left</button>
   <button type="button" class="btn btn-secondary float-end">Example Button floated right</button>
 </div>

Ratio

Usa pseudo elementos generados para hacer que un elemento mantenga la relación de aspecto de tu elección. Perfecto para manejar incrustaciones de video o presentación de diapositivas de manera responsive según el ancho del padre.

              
        <div class="ratio ratio-1x1">
          <div>1x1</div>
        </div>
        <div class="ratio ratio-4x3">
          <div>4x3</div>
        </div>
        <div class="ratio ratio-16x9">
          <div>16x9</div>
        </div>
        <div class="ratio ratio-21x9">
          <div>21x9</div>
        </div>

Streched links

Haz que se pueda hacer clic en cualquier elemento HTML o componente de Bootstrap “estirando” un enlace anidado usando CSS.

              
 <div class="card" style="width: 18rem;">
   <img src="..." class="card-img-top" alt="...">
   <div class="card-body">
     <h5 class="card-title">Card with stretched link</h5>
     <p class="card-text">Some quick example text to build on the card title and make up the bulk of the card's content.</p>
     <a href="#" class="btn btn-primary stretched-link">Go somewhere</a>
   </div>
 </div>

Text truncation

Trunca cadenas largas de texto con puntos suspensivos.

              
 <!-- Block level -->
 <div class="row">
   <div class="col-2 text-truncate">
     Este texto es bastante largo y se truncará una vez que se muestre.
   </div>
 </div>
 
 <!-- Inline level -->
 <span class="d-inline-block text-truncate" style="max-width: 150px;">
   Este texto es bastante largo y se truncará una vez que se muestre.
 </span>

Vertical rule

Utiliza el asistente de regla vertical personalizado para crear divisores verticales como el elemento <hr>.

              
 <div class="d-flex" style="height: 200px;">
   <div class="vr"></div>
 </div>
